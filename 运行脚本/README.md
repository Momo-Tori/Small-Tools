## 使用

### 源文件

将学生的源文件在文件夹`ex/student-name/`下按照题号进行排列，如在`ex/student1`下，源文件如下，按照题号名+`.c`或其他`.cpp`等的格式放置

```
1.c
2.cpp
```

### 题目输入

在`input`下放置输入文件，后缀为`.txt`

```
1.txt
2.txt
```

如果本题无需输入则不用创建输入文件

### 运行

**linux**环境下在根目录运行脚本`run.py`，在源文件同文件夹的地方生成输出文件，如`1.txt`
